# Forge2 Kanban Agent Skill

- **Stack:** Laravel (Backend) + React + Vite (Frontend) + SQLite (Database)
- **Goal:** Maintain, build, and deploy a full-stack Kanban application following Forge 2 qualifier standards.

## Rules for Agents
1. Always explain changes before making them.

3. Keep frontend (Vite/React) and backend (Laravel API) synced.
